追加 現在数 タイトル 説明
0 BtoBマーケティング.jp 「BtoBマーケティング.jp」の最新情報をお知らせします。
0 Gmail Checker Gmailの未読メールを表示します。自動更新のため常に新着メールの有無を確認することができます。
1 ToDo やるべき作業を管理するための機能です。メンバーにも共有されるため、全員で作業の進捗状況をリアルタイムに確認することができます。
0 twiリーダー 設定したキーワードでTwitterを検索し、内容を表示することができます。キーワードやハッシュタグを3つまで設定できます。会社名や会社の製品名などを設定することで、毎日の確認に役立ちます。
0 Webデータベース 項目を自由に設定してオリジナルの表を作成できます。データの追加も簡単に行えます。表計算ソフトを使う必要がないため複数人で同時に編集ができ、管理が分散しません。
0 Webページ Aipo.com内で、指定したURLのWebサイトを表示することができます。複数のアプリを配置することで、それぞれ違ったWebサイトを表示することもできます。
0 Webメール 設定したアカウントのメールが送受信できるWebメール機能です。オリジナルの署名も設定することができます。
0 Wiki 独自のWikiが作成できます。文章中への表や画像の挿入、記事の装飾もボタン一つで行うことができるため、どなたでも簡単にページの追加・編集を行えます。
0 Yahoo!ニュース Yahoo!ニュースへ掲載されているニュースをお知らせします。自動更新のため、常に最新のニュースを確認することができます。
0 つぶやき シンプルなテキストで、メッセージをやりとりすることができます。発言ごとにコメントを付けることができるため、リアルタイムな情報共有に役立ちます。
1 アドレス帳 個人のアドレス、会社情報が管理・共有できます。非公開で登録することもできるため、幅広く管理できます。
0 エクセル関数辞典 エクセル関数を、関数名で用途を検索できるほか、「合計」などの用途から逆引き検索をすることもできます。
1 スケジュール メンバー全員のスケジュール共有が行えます。設備予約や継続スケジュールの登録など詳細な設定も可能です。表示形式はブロック・日別・週別・月別・一覧から選択できます。
1 ストレージ 簡単な操作でファイル共有ができます。ドラッグ&ドロップでファイルの操作を行うことができ、どこからでもアクセスが可能です。
1 タイムカード 出勤、退勤はもちろん外出も1クリックで登録できます。ひと月ごとに自動で集計されるため計算の手間がかかりません。
1 タイムライン チャット感覚で情報共有ができ、予定やToDoの追加・編集などが時系列に表示されることで社内の動きを把握してコミュニケーションできます。
0 チャットルーム 短いメッセージでリアルタイムなやりとりを行える、昔ながらのチャット機能です。古い発言から自動で削除されます。
0 フィードリーダー お気に入りのRSS、Atomフィードを購読できます。記事が更新された際に、「あなた宛のお知らせ」に通知をすることもできます。
1 フォルダ ファイルを共有するための機能です。フォルダを階層化して管理できます。また、フォルダ毎にアクセス権限も設定できるため誰が何を閲覧できるかを管理できます。
1 ブログ 共有テーマを設定して記事を作成できます。全てのブログ記事からキーワードを入力して検索することができるため、日記のほか日報、ノウハウ共有などにも最適です。
0 マルチ検索窓 キーワードを入力して、Yahoo!、Google、Bingのいずれか、または全てで一度に検索をかけることができます。
1 ユーザー名簿 登録されているユーザー情報が一覧で表示されます。キーワードを入力して検索することもできます。
1 ワークフロー 決済や有給などの稟議・申請を簡単に行うことができます。申請経路や、テンプレートをあらかじめ設定しておくことも可能です。申請・承認はお知らせで通知されます。
1 伝言メモ メンバーを指定して連絡事項などを送ることができます。登録されているメールアドレスと携帯メールアドレスにも通知できるため、外出中のメンバーに伝言を送るのに最適です。
0 備品管理 備品を登録して補充状況を管理することができます。備品の補充状況に応じて文字色が変わります。更新も簡単です。
0 共有メモ メモをメンバーで共有することができます。メモ集を「クリップ」しておくことにより、初回に表示させたいメモ集を登録できます。
0 共有リンク リンク集をメンバー同士で共有することができます。リンク集を「クリップ」しておくことにより、初回に表示させたいリンク集を登録できます。
1 報告書 報告したいメンバーを指定して、報告書を作成できます。既読機能により、メンバーが報告書を確認したかどうかを管理することもできます。
0 安否確認 災害や緊急時に、メンバーの状況を確認することができます。各ユーザーの安否状況と出社の可否が一覧で確認できます。
1 掲示板 カテゴリを設定して、お知らせなどを掲載できます。カテゴリ毎に閲覧や返信の権限設定を行うことも可能です。キーワードを入力して検索することができます。
0 更新情報 追加や更新があった場合にその内容を時系列で表示します。 キーワードを入力して検索することができます。
0 簡易在席確認 メンバーの在席状況を確認できます。「在席」か「外出」かのステータスアイコンと変更時間、コメントが表示されるため、在席状況をすぐに確認することができます。
0 美人時計 “美人時計であなたは１分間の恋をする。" 美人時計は、美人（1,440枚の写真）と時計（1,440分）を組み合わせた、時計アプリです。 360人、1440枚の美人写真が1分ごとに時刻をお知らせし、あなたに微笑みかけます。
0 飲みイコ 飲みへのお誘い機能です。お誘いが投稿されたら「参加したい！」ボタン一つで簡単に、参加の意志が表明できます。
