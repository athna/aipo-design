追加 現在数 タイトル 説明
1 ToDo やるべき作業を管理するための機能です。メンバーにも共有されるため、全員で作業の進捗状況をリアルタイムに確認することができます。
1 Webデータベース 項目を自由に設定してオリジナルの表を作成できます。データの追加も簡単に行えます。表計算ソフトを使う必要がないため複数人で同時に編集ができ、管理が分散しません。
0 Webページ Aipoの中で表示できる簡易ウェブブラウザです。
1 Webメール 設定したアカウントのメールが送受信できるWebメール機能です。オリジナルの署名も設定することができます。
1 Wiki 独自のWikiが作成できます。文章中への表や画像の挿入、記事の装飾もボタン一つで行うことができるため、どなたでも簡単にページの追加・編集を行えます。
1 アドレス帳 個人のアドレス、会社情報が管理・共有できます。非公開で登録することもできるため、幅広く管理できます。
0 スケジュール メンバー全員のスケジュール共有が行えます。設備予約や継続スケジュールの登録など詳細な設定も可能です。表示形式はブロック・日別・週別・月別・一覧から選択できます。
0 タイムカード 出勤、退勤はもちろん外出も1クリックで登録できます。ひと月ごとに自動で集計されるため計算の手間がかかりません。
1 タイムライン チャット感覚で情報共有ができ、予定やToDoの追加・編集などが時系列に表示されることで社内の動きを把握してコミュニケーションできます。
1 フォルダ ファイルを共有するための機能です。フォルダを階層化して管理できます。また、フォルダ毎にアクセス権限も設定できるため誰が何を閲覧できるかを管理できます。
1 ブログ 共有テーマを設定して記事を作成できます。全てのブログ記事からキーワードを入力して検索することができるため、日記のほか日報、ノウハウ共有などにも最適です。
1 プロジェクト管理 プロジェクトの進捗管理ができます。複数のプロジェクトを管理でき、タスクごとの進捗状況をガントチャートで閲覧することもできます。
0 マルチ検索窓 検索用の窓です。
1 メモ帳 個人用の簡易的なメモ帳です。
1 ユーザー名簿 登録されているユーザー情報が一覧で表示されます。キーワードを入力して検索することもできます。
0 ワークフロー 決済や有給などの稟議・申請を簡単に行うことができます。申請経路や、テンプレートをあらかじめ設定しておくことも可能です。申請・承認はお知らせで通知されます。
0 伝言メモ メンバーを指定して連絡事項などを送ることができます。登録されているメールアドレスと携帯メールアドレスにも通知できるため、外出中のメンバーに伝言を送るのに最適です。
0 共有メモ ユーザー同士で共有することができるメモ集です。メモ集をクリップしておくことにより、初回に表示させたいメモ集を覚えさせておくことができます。
0 共有リンク ユーザー同士で共有することができるリンク集です。リンク集をクリップしておくことにより、初回に表示させたいリンク集を覚えさせておくことができます。
1 報告書 報告したいメンバーを指定して、報告書を作成できます。既読機能により、メンバーが報告書を確認したかどうかを管理することもできます。
0 安否確認 安否を確認できるアプリです。各自の安否と出社可否を一覧で確認できます。
1 掲示板 カテゴリを設定して、お知らせなどを掲載できます。カテゴリ毎に閲覧や返信の権限設定を行うことも可能です。キーワードを入力して検索することができます。
1 更新情報 追加や更新があった場合にその内容を時系列で表示します。 キーワードを入力して検索することができます。
